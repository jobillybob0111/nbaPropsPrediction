# NBA Prop Predictor - Mock UI

A simple mock UI for NBA player prop predictions with random output generation.

## Features

- Player name search input
- Prop type input (Points, Rebounds, Assists, etc.)
- Submit button that generates random predictions
- Displays percentage chance of hitting and missing

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

The app will open at `http://localhost:3000`

## Usage

1. Enter a player name (e.g., "LeBron James")
2. Enter a prop type (e.g., "Points", "Rebounds", "Assists")
3. Click "Get Prediction" to generate a random prediction
4. The result shows the player's percentage chance of hitting and missing the prop

## Example Output

"LeBron James has a 72% chance of hitting Points and a 28% chance of missing."
